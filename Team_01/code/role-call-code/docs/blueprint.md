# **App Name**: RoleCall

## Core Features:

- Login Page: Standard login page with username and password fields.
- Authentication: Authenticate users against a predefined set of users and roles.
- User Profile Display: After login, display the current user's name and avatar.
- Role-Based Access: Route the user to pages specific to that role.
- Avatar Generation: Use generative AI tool to create a profile picture, based on user attributes such as 'Academic Teacher'.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey authority and sophistication.
- Background color: Light purple (#EDE7F6) for a soft, professional feel.
- Accent color: Blue-purple (#3F51B5) to highlight interactive elements.
- Body and headline font: 'Inter', a sans-serif for clean readability and modern style.
- Use simple, professional icons for user roles and actions.
- Clean and organized layout for easy navigation.
- Subtle transitions on login and navigation.
